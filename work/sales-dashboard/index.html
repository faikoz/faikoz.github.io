Project · Vic Air Supplies

Sales Analytics Dashboard

Live executive view of the sales ledger, built directly on Microsoft Dynamics 365 Business Central via OData v4. MSAL-secured, with runtime $metadata schema discovery so the application is immune to vendor entity-name drift across upgrades. Single-tenant, free-tier deployment.

Business Outcome

Replaces the Power BI + Azure data-warehouse + ETL pipeline that would otherwise be required to give the leadership team a live read on sales performance.

  • JavaScript
  • MSAL OAuth 2.0
  • BC OData v4
  • Runtime $metadata discovery
  • Chart.js
  • XLSX.js
  • GitHub Pages

Screenshots redacted — synthetic blur on customer names, dollar values, and identifying fields. Layout and design are unmodified.
